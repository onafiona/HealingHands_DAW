# HealingHands_DAW
 
**Site de prezentare - Healing Hands**
Acest proiect reprezinta un site de prezentare pentru o clinica medicala fictiva. Scopul website-ului este de a prezenta pacientilor informatii despre cliniza, serviciile oferite si tarifele practicate.

**Structura website-ului**
 - index.html - pagina principala care prezinta clinica
 - services.html - include detalii despre serviciile oferite
 - prices.html - prezinta tarifele pentru fiecare doctor si serviciu
 - contact.html - pagina de contact cu un formaular de contact si o harta GoogleMaps cu locatia clinicii

**Tehnologii folosite**
 - HTML
 - CSS
 - JavaScript

**Functiile JavaScript**
 - generatePricingTables() - creaza tabelele cu preturi pentru fiecare doctor si serviciile lui
 - populateCalulatorDropdowns() - populeaza dropdown-urile cu doctorii si serviciile disponibile
 - calculateTotal() - calculeaza pretul total pentru serviciul selectat tinand cont de pretul serviciului si reducerea de 10% pentru clientii noi
